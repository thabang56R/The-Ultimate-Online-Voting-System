# 🗳️ AI-Powered Online Voting System (MERN)

A futuristic, secure, and real-time online voting platform built with the MERN stack, biometric simulations, and blockchain verification concepts.

---

## 🚀 Features

- JWT Authentication
- Election Open / Close Lifecycle
- Face Recognition Simulation
- Fingerprint Biometric Simulation
- Real-Time Voting (Socket.IO)
- Live Animated Vote Dashboard
- Blockchain Vote Verification (Simulated)
- Fully Responsive & Animated UI

---

## 🛠️ Tech Stack

**Frontend**
- React (Vite)
- Framer Motion
- Recharts
- Socket.IO Client

**Backend**
- Node.js
- Express.js
- JWT
- Socket.IO

---

## 🔐 Security Flow

1. User Authentication
2. Face Recognition Scan
3. Fingerprint Verification
4. Secure Vote Casting
5. Blockchain-style Verification
